Bacchon Ki Pathshala - Demo Flutter Project (mobile + web)

What's included:
- Splash screen with light yellow background and logo
- Home menu (Alphabets, Hindi, Numbers, Animals, Fruits, Colors)
- Sample audio assets (placeholders) and background music
- Use 'flutter pub get' then 'flutter run' to test locally
- For Codemagic: add this repo and use default workflow to build apk
